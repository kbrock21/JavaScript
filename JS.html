What happens when you add numbers and strings together?

Usually, the plus operator + sums numbers. But, if the binary + is applied to strings, it merges concatenates them: let s = "my" + "string"; alert(s); // mystring Note that if any of the operands is a string, then the other one is converted to a string too. For example:

alert( '1' + 2 ); // "12"

alert( 2 + '1' ); // "21"
See, it doesn’t matter whether the first operand is a string or the second one.
Here’s a more complex example:
alert(2 + 2 + '1' ); // "41" and not "221"

How does the Modulo (%), or Remainder, operator work?

The remainder operator %, despite its appearance, is not related to percents. The result of a % b is the remainder of the integer division of a by b.
For instance:

alert( 5 % 2 ); // 1, the remainder of 5 divided by 2 alert( 8 % 3 ); // 2, the remainder of 8 divided by 3 alert( 8 % 4 ); // 0, the remainder of 8 divided by 4

What does unary plus operator do to string representations of integers? eg. +”10”

Unary operator unary plus operator (+) can be used to convert a string representation of an integer into an actual numeric value.
When you apply the unary plus operator to a string, JavaScript attempts to convert the string to a number.
Example
let stringNumber = "42";
let numericValue = +stringNumber;
console.log(numericValue); // Output: 42 (a number)
In this case, the unary plus operator converts the string "42" into the numeric value 42.

When used with a non-numeric string:
let nonNumericString = "hello";
let numericValue = +nonNumericString;
console.log(numericValue); // Output: NaN (Not a Number)
If the string cannot be converted to a valid number, JavaScript returns NaN.
It's important to note that the unary plus operator only works with strings that represent valid numeric values.
If the string contains non-numeric characters (except for whitespace characters around the number), the result will be NaN.

let invalidString = "42px";
let numericValue = +invalidString;
console.log(numericValue); // Output: NaN
In summary, the unary plus operator in JavaScript is useful for converting string representations of integers into actual numeric values, but it's crucial to ensure that the string contains a valid numeric representation to avoid unexpected results like NaN.
